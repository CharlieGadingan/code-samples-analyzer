# Code Samples for Syntax Analyzer

This repository contains sample code files in various programming languages for testing the CodeTracker Syntax Analyzer.

## Languages Included
- Java
- Python
- C
- C++
- JavaScript
- HTML/CSS

## Structure
Each language has its own folder with:
- Basic examples (Hello World, Calculator, Student class)
- Buggy code with intentional errors to test the analyzer

## How to Use
1. Clone this repository
2. Point your CodeTracker server to this folder
3. Open the Syntax Analyzer and test each file

## File List
- Java: Object-oriented examples
- Python: Simple scripts with functions
- C: Procedural programming examples
- C++: Both procedural and OOP examples
- JavaScript: Web and Node.js style code
- HTML/CSS: Web page examples
